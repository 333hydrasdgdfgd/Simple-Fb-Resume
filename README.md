# Simple FB Resume

A clean and simple Facebook-style resume layout designed for quick presentation and easy readability.

---

## Overview
This project presents a resume in a familiar social media format, making it engaging, modern, and easy to navigate.

---

## Features
- Profile-style layout inspired by Facebook  
- Responsive and clean design  
- Organized sections (About, Skills, Experience, Education)  
- Simple and lightweight structure  

---

## Tech Stack
HTML • CSS

---

## Purpose
To create a visually appealing and easy-to-read resume using a familiar social interface.
